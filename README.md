🏋️‍♂️ FitPulse - Gym Website

A modern and responsive Gym Website built using **HTML, CSS, and JavaScript**.
This project showcases a professional fitness landing page with smooth UI, responsive layout, and interactive features.

---

🚀 Features

* 📱 Fully responsive design (mobile-friendly)
* 🍔 Hamburger menu for mobile navigation
* 🎯 Hero section with call-to-action
* 💪 Membership plans section
* 🧑‍🏫 Trainers showcase
* 📩 Contact form with validation
* ⬆️ Scroll-to-top button
* 🎨 Smooth animations and hover effects

---

🖼️ UI Sections

* Navbar (fixed + responsive)
* Hero Section
* Plans / Pricing
* Trainers Section
* Contact Form
* Footer

---

🛠️ Tech Used

* HTML5
* CSS3 (Flexbox + Grid + Media Queries)
* JavaScript (DOM + Events + Form Validation)
